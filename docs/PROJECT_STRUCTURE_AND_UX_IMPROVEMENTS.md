# 프로젝트 구조 및 UX/UI 개선 사항

## 📋 프로젝트 전체 구조

### 기술 스택
- **Frontend**: Next.js 14 (App Router), React, TypeScript, Tailwind CSS
- **Backend**: Node.js, Express, TypeScript
- **Database**: PostgreSQL
- **인증**: 치지직 OAuth 2.0 (네이버 계정)
- **상태 관리**: React Query (TanStack Query)

### 디렉토리 구조
```
cheese3/
├── backend/              # 백엔드 API 서버
│   ├── src/
│   │   ├── routes/       # API 라우트

│   │   ├── db/          # DB 연결
│   │   └── utils/       # 유틸리티
│   ├── db/

│
├── web/                  # 프론트엔드 웹앱

│   │   ├── page.tsx     # 홈 페이지
```

## 🎯 주요 기능

### 1. 인증 시스템
- **네이버 계정 로그인**: 치지직 OAuth를 통한 네이버 계정 로그인
- **자동 회원가입**: OAuth 로그인 시 자동으로 계정 생성
- **개발자 모드**: 목 데이터로 테스트 가능 (비밀번호 보호)

### 2. 사용자 역할
- **시청자 (viewer)**: 크리에이터 팔로우, 메시지 전송, 후원
- **크리에이터 (creator)**: 채널 관리, 메시지 관리, 후원 확인
- **관리자 (admin)**: 시스템 관리

### 3. 핵심 기능
- **후원 시스템**: 치즈 후원 + 메시지 전송
- **메시징**: 비공개 DM, 공개 피드, 크리에이터 공개 메시지
- **팔로우 시스템**: 크리에이터 팔로우/언팔로우
- **검색**: 크리에이터 검색
- **인기 크리에이터**: 팔로워 수 기준 정렬

## 🎨 UX/UI 개선 사항

### ✅ 완료된 개선
1. **홈 페이지 개선**
   - 비밀번호 보호 개발자 모드 버튼 추가
   - 둘러보기 버튼 추가
   - 명확한 버튼 배치 및 스타일

2. **둘러보기 페이지 생성**
   - 로그인 없이 인기 크리에이터 목록 확인 가능

## 🎯 즉시 적용 가능한 개선 사항

### 1. 홈 페이지 버튼 순서 및 스타일
```tsx
// 권장 순서:
1. 네이버로 시작하기 (주요 CTA - 큰 버튼, 초록색)
2. 둘러보기 (보조 CTA - 중간 크기, 회색)
3. 개발자 모드 (작은 텍스트 링크 또는 하단 배치)
```

### 2. 둘러보기 페이지 개선
- 크리에이터 카드에 더 많은 정보 표시
- 그리드 레이아웃 최적화 (2열 → 3열)
- 로딩 스켈레톤 추가

### 3. 전역 네비게이션
- 상단 고정 네비게이션 바
- 로그인 상태 표시
- 알림 배지

### 4. 메시지 UI 개선
- 메시지 버블 디자인
- 읽음/안읽음 표시
- 시간 표시 개선

## 📱 반응형 디자인 가이드

### 브레이크포인트
- **모바일**: < 640px
- **태블릿**: 640px - 1024px
- **데스크톱**: > 1024px

### 모바일 우선 설계
- 터치 친화적 버튼 크기 (최소 44x44px)
- 충분한 간격 (최소 8px)
- 가독성 있는 폰트 크기 (최소 14px)

## 🎨 디자인 시스템

### 색상 팔레트
- **주요 색상**: 네이버 그린 (#03C75A)
- **강조 색상**: 블루 (#3B82F6)
- **배경**: 다크 모드 (neutral-950 ~ neutral-900)
- **텍스트**: neutral-100 ~ neutral-500

### 타이포그래피
- **제목**: font-bold, text-2xl ~ text-4xl
- **본문**: font-normal, text-sm ~ text-base
- **캡션**: text-xs, text-neutral-500

### 간격 시스템
- **기본 간격**: 4px 단위 (space-y-4, gap-4 등)
- **섹션 간격**: space-y-6 ~ space-y-8
- **패딩**: p-4 ~ p-8

# 미구현 페이지 분류 및 구현 계획

## 현재 구현 상태

### ✅ 완료된 페이지
- 인증 및 온보딩
- 메인 대시보드
- 대화방 목록
- 크리에이터 검색
- 채널 페이지 및 메신저
- 후원 페이지
- 크리에이터 대시보드
- 크리에이터 메시지 관리
- 크리에이터 설정
- 프로필 설정 (기본)

## 미구현 페이지 분류

### 1. 마이페이지 개선 (우선순위: 높음)

#### 1.1 내 활동 내역 페이지
**경로**: `/app/my/activity`
**기능**:
- 내가 보낸 메시지 목록
- 내가 한 후원 내역
- 내가 팔로우한 크리에이터 목록
- 내가 받은 알림 내역

#### 1.2 알림 센터
**경로**: `/app/notifications`
**기능**:
- 읽지 않은 알림 수 표시
- 알림 목록 (메시지, 후원, 팔로우 등)
- 알림 읽음 처리
- 알림 설정

#### 1.3 프로필 개선
**경로**: `/app/profile` (기존 페이지 개선)
**추가 기능**:
- 프로필 이미지 업로드
- 배경 이미지 설정
- 소개글 작성
- 통계 요약 (내가 보낸 메시지 수, 후원 총액 등)

### 2. 관리자 페이지 (우선순위: 중간)

#### 2.1 관리자 대시보드
**경로**: `/app/admin`
**기능**:
- 전체 통계 (사용자 수, 메시지 수, 후원 총액)
- 최근 활동 요약
- 시스템 상태

#### 2.2 유저 관리
**경로**: `/app/admin/users`
**기능**:
- 유저 목록 조회
- 유저 검색
- 유저 상세 정보
- 유저 차단/해제
- 권한 관리

#### 2.3 채널 관리
**경로**: `/app/admin/channels`
**기능**:
- 채널 목록 조회
- 채널 검색
- 채널 상세 정보
- 채널 승인/거부

#### 2.4 메시지 모더레이션
**경로**: `/app/admin/messages`
**기능**:
- 신고된 메시지 목록
- 메시지 삭제
- 유저 차단
- 스팸 메시지 필터링

### 3. 기타 페이지 (우선순위: 낮음)

#### 3.1 도움말/FAQ
**경로**: `/help` 또는 `/faq`
**기능**:
- 자주 묻는 질문 목록
- 검색 기능
- 카테고리별 분류

#### 3.2 문의하기
**경로**: `/contact`
**기능**:
- 문의 양식
- 문의 내역 조회
- 답변 확인

#### 3.3 이용약관 및 개인정보처리방침
**경로**: `/terms`, `/privacy`
**기능**:
- 정적 페이지
- 최신 버전 표시

## 구현 로드맵

### Phase 1.3 - 마이페이지 개선 (1주)
1. 내 활동 내역 페이지
2. 알림 센터
3. 프로필 개선 (이미지 업로드 등)

### Phase 2.1 - 관리자 페이지 (2주)
1. 관리자 대시보드
2. 유저 관리
3. 채널 관리
4. 메시지 모더레이션

### Phase 3 - 부가 기능 (1주)
1. 도움말/FAQ
2. 문의하기
3. 이용약관 및 개인정보처리방침

## 구현 원칙

1. **개발 모드 우선**: 모든 기능은 Mock 데이터로 먼저 구현
2. **점진적 개선**: Phase별로 기능 추가
3. **테스트 가능성**: 실제 백엔드 없이도 테스트 가능

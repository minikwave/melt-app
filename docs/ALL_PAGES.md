# Melt 전체 페이지 목록

## 📍 페이지 구조

```
/
├── / (홈)
├── /auth/
│   ├── /naver (네이버 로그인)
│   ├── /login (리다이렉트)
│   └── /chzzk/callback (OAuth 콜백)
├── /onboarding/
│   ├── / (역할 선택)
│   └── /creator-setup (스트리머 설정)
├── /dev/
│   └── /login (개발 모드 로그인)
├── /app/
│   ├── / (메인 대시보드)
│   ├── /profile (프로필 설정)
│   ├── /my/activity (내 활동 내역)
│   ├── /notifications (알림 센터)
│   ├── /conversations (대화방 목록)
│   ├── /search (크리에이터 검색)
│   ├── /channels/
│   │   ├── / (채널 검색)
│   │   └── /[id]/
│   │       ├── / (채널 메신저)
│   │       └── /donate/
│   │           ├── / (후원 페이지)
│   │           └── /complete (후원 완료)
│   └── /creator/
│       ├── /dashboard (대시보드)
│       ├── /messages (메시지 관리)
│       └── /settings (채널 설정)
├── /app/admin/
│   ├── / (관리자 대시보드)
│   ├── /users (유저 관리)
│   ├── /channels (채널 관리)
│   └── /messages (메시지 모더레이션)
├── /help (도움말/FAQ)
├── /contact/
│   ├── / (문의하기)
│   └── /history (문의 내역)
├── /terms (이용약관)
└── /privacy (개인정보처리방침)
```

## 📄 페이지별 상세 정보

### 인증 및 온보딩

#### `/` - 홈 페이지
- **기능**: 서비스 소개 및 로그인 옵션
- **접근**: 공개
- **링크**: 네이버 로그인, 개발 모드, 둘러보기

#### `/auth/naver` - 네이버 로그인
- **기능**: 네이버 OAuth를 통한 치지직 로그인 시작
- **접근**: 공개
- **리다이렉트**: 치지직 OAuth → `/auth/chzzk/callback`

#### `/auth/chzzk/callback` - OAuth 콜백
- **기능**: 치지직 OAuth 콜백 처리
- **접근**: 인증 필요
- **리다이렉트**: `/onboarding` 또는 `/app`

#### `/dev/login` - 개발 모드 로그인
- **기능**: Mock 데이터로 테스트
- **접근**: 공개
- **사용자**: 더미 유저 선택 가능

#### `/onboarding` - 역할 선택
- **기능**: 시청자/크리에이터 역할 선택
- **접근**: 인증 필요
- **리다이렉트**: 크리에이터 → `/onboarding/creator-setup`, 시청자 → `/app`

#### `/onboarding/creator-setup` - 크리에이터 설정
- **기능**: 채널 ID, 후원 링크 등 초기 설정
- **접근**: 크리에이터만
- **단계**: 3단계 설정 플로우

### 메인 앱

#### `/app` - 메인 대시보드
- **기능**: 빠른 액세스 메뉴, 인기 크리에이터 목록
- **접근**: 인증 필요
- **메뉴**: 알림, 대화방, 검색, 프로필, 도움말

#### `/app/profile` - 프로필 설정
- **기능**: 프로필 이미지, 닉네임, 소개글 관리
- **접근**: 인증 필요
- **기능**: 이미지 업로드, 닉네임 변경, 소개글 작성

#### `/app/my/activity` - 내 활동 내역
- **기능**: 메시지, 후원, 팔로우 내역 조회
- **접근**: 인증 필요
- **탭**: 메시지, 후원, 팔로우

#### `/app/notifications` - 알림 센터
- **기능**: 알림 목록 및 읽음 처리
- **접근**: 인증 필요
- **기능**: 읽지 않은 알림 수 표시, 알림 타입별 구분

#### `/app/conversations` - 대화방 목록
- **기능**: 팔로우한 크리에이터와의 대화 목록
- **접근**: 시청자만
- **기능**: 읽지 않은 메시지 수 표시

#### `/app/search` - 크리에이터 검색
- **기능**: 크리에이터 검색 및 팔로우
- **접근**: 시청자만
- **기능**: 실시간 검색, 팔로우/언팔로우

#### `/app/channels/[id]` - 채널 메신저
- **기능**: 크리에이터와의 메시지 교환
- **접근**: 인증 필요
- **기능**: 공개 메시지, DM, 후원 메시지 표시

#### `/app/channels/[id]/donate` - 후원 페이지
- **기능**: 치즈 후원 시작
- **접근**: 인증 필요
- **리다이렉트**: 치지직 후원 페이지

#### `/app/channels/[id]/donate/complete` - 후원 완료
- **기능**: 후원 완료 후 메시지 등록
- **접근**: 인증 필요
- **기능**: 공개 메시지 입력 및 등록

### 크리에이터

#### `/app/creator/dashboard` - 크리에이터 대시보드
- **기능**: 후원 통계 및 인박스 요약
- **접근**: 크리에이터만
- **기능**: 기간별 통계, 미확정 후원, DM 목록

#### `/app/creator/messages` - 메시지 관리
- **기능**: DM 및 미확정 후원 관리
- **접근**: 크리에이터만
- **기능**: 답장, RT, 후원 확정

#### `/app/creator/settings` - 채널 설정
- **기능**: 채널 정보 및 후원 링크 관리
- **접근**: 크리에이터만
- **기능**: 채널 URL, 후원 링크, 충전 링크 설정

### 관리자

#### `/app/admin` - 관리자 대시보드
- **기능**: 전체 통계 및 활동 요약
- **접근**: 관리자만
- **기능**: 사용자 수, 메시지 수, 후원 통계

#### `/app/admin/users` - 유저 관리
- **기능**: 유저 목록 및 검색
- **접근**: 관리자만
- **기능**: 유저 검색, 역할 표시

#### `/app/admin/channels` - 채널 관리
- **기능**: 채널 목록 및 검색
- **접근**: 관리자만
- **기능**: 채널 검색, 정보 표시

#### `/app/admin/messages` - 메시지 모더레이션
- **기능**: 신고된 메시지 관리
- **접근**: 관리자만
- **기능**: 신고 내역, 메시지 삭제

### 기타

#### `/help` - 도움말/FAQ
- **기능**: 자주 묻는 질문 및 검색
- **접근**: 공개
- **기능**: 카테고리별 필터, 검색

#### `/contact` - 문의하기
- **기능**: 문의 양식 제출
- **접근**: 공개
- **기능**: 카테고리 선택, 문의 작성

#### `/contact/history` - 문의 내역
- **기능**: 이전 문의 내역 조회
- **접근**: 인증 필요
- **기능**: 문의 상태, 답변 확인

#### `/terms` - 이용약관
- **기능**: 이용약관 내용 표시
- **접근**: 공개
- **내용**: 서비스 이용 약관

#### `/privacy` - 개인정보처리방침
- **기능**: 개인정보처리방침 내용 표시
- **접근**: 공개
- **내용**: 개인정보 처리 방침

## 🔗 페이지 간 네비게이션

### 주요 플로우

1. **로그인 플로우**
   ```
   / → /auth/naver → /auth/chzzk/callback → /onboarding → /app
   ```

2. **시청자 플로우**
   ```
   /app → /app/conversations → /app/channels/[id] → /app/channels/[id]/donate
   ```

3. **크리에이터 플로우**
   ```
   /app → /app/creator/dashboard → /app/creator/messages → /app/creator/settings
   ```

4. **프로필 플로우**
   ```
   /app → /app/profile → /app/my/activity → /app/notifications
   ```

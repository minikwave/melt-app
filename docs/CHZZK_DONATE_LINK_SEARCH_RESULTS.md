# 치지직 후원 링크 웹 검색 결과

## 결론: 웹 검색만으로는 후원 링크를 찾을 수 없습니다

### 확인된 사실

1. **치지직 공식 Open API 문서 존재**
   - 문서 위치: https://chzzk.gitbook.io/chzzk
   - 공개된 API 범위:
     - ✅ 인증/세션
     - ✅ 채널 정보 조회
     - ✅ 라이브 정보
     - ✅ 채팅
     - ✅ 드롭스(프로모션)
     - ❌ **후원(치즈) 링크/딥링크는 문서에 없음**

2. **커뮤니티/비공식 라이브러리**
   - GitHub의 여러 치지직 라이브러리 확인
   - 채널, 라이브, 채팅 기능은 제공
   - **후원 링크 생성/추출 기능은 없음**

3. **치지직 개발자 포털**
   - https://developers.chzzk.naver.com/
   - 공개 문서에 후원 링크 스펙 없음

## 왜 웹 검색으로 찾을 수 없는가?

1. **공식 문서에 미기재**
   - 치지직 Open API 문서에 후원 링크 관련 항목이 없음
   - 채널 정보 API 응답에도 후원 링크 필드가 없음

2. **비공개/내부 API 가능성**
   - 후원 링크가 내부 스튜디오/파트너 도구에서만 노출될 수 있음
   - 공개 API가 아닐 가능성

3. **동적 생성**
   - 후원 링크가 페이지 로드 시 JavaScript로 동적 생성될 수 있음
   - 고정 URL 패턴이 아닐 수 있음

## 대안: 실제 확인 방법

### 방법 1: 브라우저 개발자 도구 (필수)

**가장 확실한 방법**:
1. Elements 탭에서 버튼 요소 확인
2. Network 탭에서 실제 요청 추적
3. Console에서 JavaScript 코드 확인

**상세 가이드**: [CHZZK_DONATE_INSPECTION_STEPS.md](./CHZZK_DONATE_INSPECTION_STEPS.md)

### 방법 2: 치지직 고객센터 문의

1. **치지직 고객센터**
   - 후원 링크/딥링크 공식 스펙 문의
   - 개발자 포털 문의

2. **치지직 개발자 포털 문의**
   - https://developers.chzzk.naver.com/
   - API 관련 문의

### 방법 3: 세션 이벤트로 우회 확인

치지직 API 문서에 DONATION 이벤트가 있다면:
- 후원 발생을 감지하는 방식으로 우회 확인 가능
- 하지만 링크 자체는 여전히 개발자 도구로 확인 필요

## 확인 가능한 정보 (웹 검색으로)

### ✅ 공식 API 문서
- **위치**: https://chzzk.gitbook.io/chzzk
- **내용**:
  - 인증 방법
  - 채널 정보 조회 API
  - 라이브 정보 API
  - 채팅 API
  - 세션 이벤트 (DONATION 이벤트 포함)

### ✅ 기본 정보
- 치지직은 네이버 산하 서비스
- 치즈는 인앱 결제 항목
- 후원 기능 존재 (하지만 링크 스펙은 공개되지 않음)

## 권장 접근 방법

### 1단계: 공식 API 문서 확인
```
https://chzzk.gitbook.io/chzzk/chzzk-api/channel
```
- 채널 정보 API 응답 구조 확인
- 혹시 모를 후원 관련 필드 확인

### 2단계: 브라우저 개발자 도구로 실제 확인
- Elements 탭에서 버튼 확인
- Network 탭에서 요청 추적
- **이 방법이 가장 확실함**

### 3단계: 확인된 정보를 Melt에 적용
- 확인된 링크 형식을 Melt 코드에 반영
- 크리에이터 설정 페이지에서 등록 가능하도록 구현

## 결론

**웹 검색만으로는 후원 링크를 찾을 수 없습니다.**

따라서:
1. ✅ **브라우저 개발자 도구로 실제 확인** (필수)
2. ✅ **치지직 고객센터/개발자 포털 문의** (선택)
3. ✅ **확인된 정보를 Melt에 적용**

현재 Melt는 이미 유연하게 후원 링크를 처리할 수 있도록 구현되어 있으므로, 확인된 링크를 크리에이터 설정 페이지에서 등록하면 됩니다.
